    Working Style

    1. Scope

    เลือกรูปแบบงาน จำนวนตัวละคร ฉาก และสิทธิ์ใช้งานให้ชัดก่อนประเมินราคา

    2. Sketch

    ส่งร่างตามวันที่ตกลง ลูกค้าแก้รอบใหญ่ตามจำนวนที่ระบุในบรีฟ

    3. Delivery

    ส่งไฟล์ตาม scope งาน หากต้องการ source file หรือแยกพาร์ตให้เลือกตั้งแต่ต้น

    Terms & Flow

    1. ราคาที่เว็บคำนวณเป็นราคาเริ่มต้น ใช้สำหรับคัดกรอง requirement ก่อนตกลงงานจริง
    2. เริ่มงานหลังยืนยัน scope และชำระเงินตามเงื่อนไขที่ตกลง
    3. แก้ใหญ่เพิ่มเติมคิดเพิ่มตามจำนวนรอบที่เลือก ส่วนแก้เล็กน้อยในขั้นตอนร่างขึ้นกับข้อตกลง
    4. สิทธิ์เชิงพาณิชย์ต้องเลือกในระบบก่อนใช้งานกับสินค้า โปรโมชัน หรือรายได้โดยตรง
